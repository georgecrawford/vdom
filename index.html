The red outlined area below has a DOM view generated by FTFruitMachine. Clicking the button will regenerate the view with different colour values, and random updates to the text header. The diff between the current and new view is calculated, and a patch is applied to update the DOM. Inspecting the DOM mutation events, you should see that the only changes to the DOM are the style attribute and the inner text of the header. Other elements are untouched.
